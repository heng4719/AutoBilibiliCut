# 视频切片工具开发文档

## 1. 项目目标

开发一个 Windows 客户端程序，用于选择本地视频文件，读取视频基础信息，并按照用户输入的时间范围批量导出视频切片。

## 2. 技术选型

- 开发语言：C#
- 框架：.NET 8
- 客户端界面：WPF
- 视频处理：FFmpeg / FFprobe

## 3. 核心功能

1. 选择本地视频文件
2. 读取并显示视频信息
   - 文件名
   - 文件大小
   - 视频时长
   - 文件路径
3. 添加多个切片任务
   - 开始时间
   - 结束时间
   - 切片名称
4. 批量导出切片到指定目录

## 4. 界面规划

程序采用单窗口布局，界面尽量简洁易用：

- 顶部：视频文件选择区域
- 中部：视频信息展示区域
- 下方：切片列表编辑区域
- 底部：输出目录与导出操作区域

## 5. 开发阶段

1. 搭建项目与主界面
2. 实现视频信息读取
3. 实现切片列表编辑
4. 实现批量导出功能

## 6. 说明

开发过程采用分步协作方式，每完成一个可检查任务后暂停，由人工检查确认后再继续下一步。

## 7. FFmpeg 集成

项目使用自带的 FFmpeg，不依赖系统全局安装。

文件放置目录：

- `tools/ffmpeg/bin/ffmpeg.exe`
- `tools/ffmpeg/bin/ffprobe.exe`

程序运行时会优先从上述目录读取这两个文件。
