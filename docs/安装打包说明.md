# 安装打包说明

## 1. 说明

项目已经配置好发布与安装脚本，发布结果会自动包含这些本地工具：

- `tools/ffmpeg/bin/ffmpeg.exe`
- `tools/ffmpeg/bin/ffprobe.exe`
- `tools/yt-dlp/yt-dlp.exe`

安装后程序仍然优先使用自带工具，不依赖系统全局安装。

## 2. 前置准备

需要先安装：

1. `.NET 8 SDK`
2. `Inno Setup 6`

## 3. 生成发布目录

在项目根目录执行：

```powershell
.\scripts\publish.ps1
```

默认会生成自包含的 `win-x64` 发布目录，输出位置：

- `artifacts/publish/win-x64/`

如果你想生成依赖本机 .NET 的较小体积版本：

```powershell
.\scripts\publish.ps1 -FrameworkDependent
```

## 4. 生成安装包

在项目根目录执行：

```powershell
.\scripts\build-installer.ps1
```

脚本会先检查发布目录，不存在时会自动先执行发布，然后调用 Inno Setup 生成安装包。

安装包输出位置：

- `artifacts/installer/`

## 5. 产物说明

默认生成的安装程序为：

- `videoCut-setup-1.0.0.exe`

安装包内容包含：

- 主程序 `videoCut.exe`
- 程序依赖文件
- `tools/ffmpeg/bin/`
- `tools/yt-dlp/`

## 6. 可修改项

如果后续要调整安装信息，可以修改：

- 安装脚本：[videoCut.iss](/F:/WorkSpace/videoCut/installer/videoCut.iss)
- 发布脚本：[publish.ps1](/F:/WorkSpace/videoCut/scripts/publish.ps1)
- 安装脚本入口：[build-installer.ps1](/F:/WorkSpace/videoCut/scripts/build-installer.ps1)
